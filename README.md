# Description

This is a mini-project based on an assignment from The Odin Project. The initial requirement was to make a console-based game but later in the curriculum another task required rebuilding it as a UI-based one, implementing DOM manipulation and event handling. 

[ Live Demo ](https://cumhurbabaoglu.github.io/javascript-rock-paper-scissors/)

There is another version of the game, which I built earlier in the curriculum, where the user is required to input their choice in contrast to this version that is played by clicking buttons.

[ Input Version Repository ](https://github.com/cumhurbabaoglu/rock-paper-scissors-dom)
[ Input Version Live Demo ](https://cumhurbabaoglu.github.io/rock-paper-scissors-dom/)

# Features
- First to 50 points wins
- Score tracking across rounds
- Button-based gameplay
- Round counter display

# Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

# What I Practiced & Learned
- DOM manipulation
- Event handling (pointer and click events)
- Working with NodeList and the forEach method
- Game and state reset logic implementation
